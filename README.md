# AI 기반 Play Store 앱 비교·추천 서비스

Google Play 스토어의 여러 앱을 하나씩 비교해야 하는 번거로움을 줄이기 위해,
리뷰 기반 키워드 분석과 LLM을 활용해 앱 자동 비교표를 생성하는 서비스입니다.

## 프로젝트 목표

사용자가 원하는 기능의 앱을 더 빠르고 정확하게 찾을 수 있도록 지원

Google Play 리뷰 데이터를 기반으로 앱 평가 기준을 자동 생성

### 주요 기능
1. Google Play 앱 리뷰 크롤링

google-play-scraper 라이브러리를 사용해
스도쿠 앱 상위 10개 리뷰(최신순)를 수집

리뷰 내용, 평점, 날짜 등 저장 (CSV)

2. 리뷰 전처리 & 키워드 분석

한국어 텍스트 정제(cleaning)

광고 / 난이도 / 과금 / 오류 / UI / 기능 다양성 등
그룹 키워드 기반 리뷰 빈도 분석

리뷰별 1회 카운트 방식으로 과도한 편향방지
